[학습주제] 4-1-1.운영체제(180~186p.)

[학습목표]

  - 운영체제의 개념과 기능을 이해할 수 있다.

1. 운영체제(Operating System, OS)

  가. 개념

    1) 컴퓨터의 전반적인 부분을 관리함.

    2) 하드웨어와 소프트웨어를 관리하여 컴퓨터가 효율적으로 작동할 수 있게 함.

    3) 사용자가 컴퓨팅 시스템을 쉽고 편리하게 사용할 수 있는 환경을 제공함.

  나. 대표적인 운영체제

    1) 안드로이드: Google 사에서 linux 커널을 기반으로 만든 운영체제, 무료 라이센스

    2) iOS: Apple사에서 생산하는 모바일 기기에 탑재, MacOS 기반으로 만들어져 호환성과

                보안성이 뛰어나며 직관적인 사용자 인터페이스를 가짐.

    3) 윈도: Microsoft사에서 만든 데스크톱에서 가장 많이 쓰이는 운영체제로

    4) 맥: Apple사에서 만든 데스크톱에서 Windows 다음으로 많이 쓰이는 운영체제로 초기에

              GUI를 빠르게 적용하여 편리성을 줌.

    5) 리눅스: 유닉스 기반 컴퓨터용 공개 운영체제로 무료로 배포됨.

2. 운영체제의 기능

  가. 사용자 인터페이스

  운영체제는 사용자와 컴퓨팅 시스템 간 의사소통 수단인 사용자 인터페이스(UI)를 제공한다.

  1. CUI(명령줄 인터페이스): 사용자가 직접 키보드를 이용해 특정 명령어를 실행하는 방식
  2. GUI(그래픽 사용자 인터페이스): 아이콘 등의 그래픽을 통해 시각적으로 알기 쉽게 표현한 방식
  3. NUI(내추럴 사용자 인터페이스): 사람의 손짓이나 음성, 시선 등을 자연스럽게 사용하는 방식

  나. 자원 관리: 운영체제가 한정된 자원을 여러 프로그램이 효율적으로 사용할 수 있도록

                        분배하거나 회수

  다. 보안 관리: 자원과 프로그램에 대한 접근을 관리하고 보호하는 기능 제공

                       (예: 로그인 암호 설정, 계정에 대한 접근 권한 설정)

  라. 파일 관리: 파일의 생성, 수정, 삭제, 복사 등이 이루어지는 과정을 관리하고, 파일의 저장위치

                        를 결정하거나 저장된 파일의 위치를 검색할 수 있도록 지원

  마. 입출력 장치 관리: 키보드, 마우스, 프린터, 모니터 등의 입출력 장치를 사용자나 프로그램이 사용할 수 있도록 지원하며 장치가 추가될 때에는 장치 드라이버를 설치하고 이용하여 관리

  바. 네트워크 관리: 컴퓨팅 시스템의 내부 설정, 인터넷 사용량 및 연결 상태, 통신을 위한 규칙,

                              유선 또는 무선 등의 인터넷 연결 방식 등을 관리
