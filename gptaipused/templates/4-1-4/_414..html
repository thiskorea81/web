 [학습주제] 4-1-4. 네트워크 환경 설정(194~200p.)

[학습목표]

  - 컴퓨팅 시스템의 네트워크 환경을 설정할 수 있다.

1. IP 주소

  가.개념

    1) 인터넷에 연결된 컴퓨팅 시스템끼리 중복되지 않는 유일하고 고유한 번호

    2) IPv4
     - 총 32비트로 8비트씩 십진수로 변환하여 마침표(.)로 구분

    3) IPv6: IPv4의 주소 부족 문제를 해결하기 위해 제정

      - 총 128비트로 32비트씩 십육진수로 변환하여 콜론(:)으로 구분

방법

1. 네이버에 ‘내 IP’ 검색-IPv4주소만 확인 가능

2. 명령 프롬프트(cmd) 창에 ipconfig 명령 입력

 나.공인 IP주소와 사설 IP주소

    1) 사설 IP 주소

      -내부적으로 서로 구별할 수 있는 IP 주소

      -내부 네트워크의 관리자로부터 부여받아 작은 규모의 네트워크에서 사용

    2) 공인 IP 주소

      - 내부 네트워크의 컴퓨팅 시스템들이 공유하는 하나의 IP 주소

      - 공인된 기관으로부터 부여받아 부여받아 외부 네트워크와의 연결이 필요할 때  사용

 

다. 규모에 따른 네트워크의 분류

작은 원부터 개인 영역 네트워크, 근거리 네트워크, 중거리(도시 지역) 네트워크, 광역 네트워크

2. 네트워크의 환경 설정

  가. 고정IP 방식과 자동 IP 방식

     1) 고정IP 방식

       - 컴퓨팅 시스템의 IP주소를 수동으로 정하는 것

       - 기업이나 학교에서 주로 사용

     2) 자동IP 방식

       - 컴퓨팅 시스템이 접속할 때마다 서버가 자동으로 사용 가능한 IP주소를 부여하는 것

       - 가정에서 IP 주소 공유기를 사용할 때 많이 사용

  나. 네트워크 환경 설정 방법

     1) 네트워크 인터페이스 카드(NIC)에 인터넷 케이블을 연결한다.(물리적 연결 단계)

     2) 네트워크 설정 화면을 띄운다.

     3) IP 주소와 부분망(subnet) 마스크, 기본 게이트웨이, 도메인 이름 서버를 설정한다.

     4) 인터넷 연결을 확인한다.
